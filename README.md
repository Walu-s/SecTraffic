# SecTraffic by Walum
**Хочешь разыграть друга? Тогда тебе сюда, этот софт предназначен для создания нагрузки на раздаваемую сеть и поедания трафика.**

  • platform

# Установка
Для Linux, termux 

    git clone https://github.com/Walu-s/SecTraffic
    cd SecTraffic
    pip install -r import.txt
    python SecTraffic.py

**Софт построена на вызове функций и отправке HTTP-запросов**

# Поддержка
Работает на Linux, Windows and Termux!
